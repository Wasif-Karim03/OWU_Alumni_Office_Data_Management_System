How to Use Data Management

📁 Creating Event Files
  1. Go to the "Create New File" tab
  2. Enter a descriptive name for your event file
  3. Click "Create File" to start a new collection
➕ Adding Events
  1. Open an existing event file
  2. Click "Add New Event"
  3. Fill in the event details:
    • Date: Event date
    • Event Name: Name of the event
    • Event Income: Revenue from the event
    • All Incurred Expenses: Food, beverage, rental costs
    • Underwritten: Sponsored amounts
  4. Profit/Loss is automatically calculated
✏️ Managing Your Data
  • Edit: Click the edit button to modify event details
  • Delete: Remove individual events or entire files
  • Export: Download your data as Excel files
  • Visualize: Use the main visualizer to create charts
💡 Tips
  • Use descriptive file names (e.g., "Fall 2024 Alumni Events")
  • Keep event names consistent and clear
  • Enter all amounts as positive numbers (expenses will be calculated as negative)
  • Export your data regularly for backup
